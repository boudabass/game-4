PRD Complet : Cerveau Numérique Cognitif
Version : 1.0 | Date : 11 janvier 2026 | Auteur : Ton symbiote conceptuel

1. Mission du Produit
Objectif fondamental : Créer un tableau blanc cognitif universel où la pensée humaine s'exprime via un réseau neuronal vivant (neurones + synapses plastiques), 100% fonctionnel sans IA.

Problème résolu : Fragmentation cognitive des outils actuels. Unification de la mémoire, du raisonnement présent et des projections futures dans un espace visuel continu.

Utilisateur cible : Tout le monde — interface instinctive (1 clic = 1 action), mobile-first.

2. Concepts Fondamentaux
Élément	Nature	Rôle
Neurone	Mémoire inerte	Contenu brut (texte/image) + métadonnées passives
Synapse	Connexion vivante	Transporte signaux, plasticité PLT/courte terme, paramètres dynamiques
Raisonnement	Flux présent	Tableau blanc infini = conscience cognitive active
Mémoire	Réservoir neuronal	Neurones dormants, tiroir repliable haut écran
Rêve (IA)	Flux parallèle	Optionnel, projection alternative non-intrusive

Panneau Neurone : Contenus Monotypiques
Principe fondamental : Chaque neurone = UN SEUL type de contenu. Le panneau affiche exclusivement ce type, en full écran (mobile) ou panneau droit (desktop), avec édition native.

Vidéo	Lecteur vidéo intégré (lien URL)	URL modifiable	YouTube/Vimeo/etc.
Tableau	Tableau Markdown editable	Ajout/suppression lignes/colonnes	Manuel
Template	Formulaire guidé prédéfini	Champs spécifiques	Système

PRD Impact
Cette monotypie purifie l'architecture : chaque neurone est un atome cognitif pur, spécialisé mais interchangeable. Le panneau devient un rendu natif adaptatif, garantissant cohérence visuelle et performance.
